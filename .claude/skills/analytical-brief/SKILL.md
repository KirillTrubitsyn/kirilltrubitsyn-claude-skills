# Аналитическая справка

Skill для создания профессиональных аналитических справок в стиле корпоративных юридических документов.

## Стандарты форматирования

Данный скилл использует единые стандарты форматирования. См. файл `formatting_standards.md` в корне папки skills/user.

### Ключевые параметры

| Элемент | Значение |
|---------|----------|
| Цвет заголовков | `#233264` (тёмно-синий) |
| Шапка таблицы | фон `#233264`, текст белый |
| Шрифт | Times New Roman |
| Размер заголовка документа | 16pt Bold |
| Размер заголовков разделов | 14pt Bold |
| Размер текста | 12pt |
| Поля | левое 3 см, остальные 2 см |
| Межстрочный интервал | 1.15 |

## Структура документа

1. **Заголовок** — крупный, по центру, цвет #233264: «АНАЛИТИЧЕСКАЯ СПРАВКА» или «ЗАКЛЮЧЕНИЕ»
2. **Подзаголовок** — тема справки, по центру, обычным шрифтом
3. **Дата** — по центру, курсив
4. **Нумерованные разделы** — заголовки цвета #233264
5. **Заключение/Выводы** — итоговый раздел

## Стиль изложения

- Профессиональный юридический язык без эмоциональной окраски
- Убедительная аргументация через факты и логику
- Структура параграфа: тезис → аргументация → вывод
- Сложные вопросы объясняются доступно
- Избегать тире где возможно (заменять на двоеточия, точки с запятой)

## Форматирование элементов

### Таблицы
- Шапка: тёмно-синий фон (#233264), белый текст, полужирный
- Данные: белый фон, шрифт 11pt
- Стиль: Table Grid

### Списки
- Маркированные: стиль List Bullet (чёрные точки)
- Нумерованные подпункты: формат 1) 2) 3) с отступом 1.5 см

### Выделение bold
Выделять:
- Ключевой вывод параграфа
- Критически важные факты и цифры
- Правовые позиции и нормы
- Предупреждения о рисках

НЕ выделять:
- Обычные факты и описания
- Даты и номера дел

## Создание документа

Использовать шаблон `scripts/create_brief.py` из данного скилла.

```python
# Пример использования
exec(open('/mnt/skills/user/analytical-brief/scripts/create_brief.py').read())

# Создание документа
add_title('АНАЛИТИЧЕСКАЯ СПРАВКА')
add_subtitle('О предмете анализа')
add_date('02 февраля 2026 г.')

add_section_heading('1. Название раздела')
add_para('Текст параграфа.')
add_para('Текст с |||выделением важной части|||.', [1])

add_bullet('Пункт маркированного списка')

data = [
    ['Показатель', 'Значение'],
    ['Строка 1', '100'],
]
add_table(data)

doc.save('/mnt/user-data/outputs/document.docx')
```

## Типовая структура разделов

### Для судебных споров
1. Существо/обстоятельства спора
2. Позиции сторон
3. Правовая оценка / Анализ
4. Риски и последствия
5. Выводы / Рекомендации

### Для корпоративных вопросов
1. Предмет анализа
2. Фактические обстоятельства
3. Правовое регулирование
4. Анализ вариантов
5. Выводы и рекомендации

### Для стратегических документов
1. Стратегические цели
2. План достижения целей
3. Анализ последствий (налоговые, правовые)
4. Выводы и рекомендации
