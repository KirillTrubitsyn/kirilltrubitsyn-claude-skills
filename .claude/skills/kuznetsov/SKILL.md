# Навык: Стиль Кузнецова

## Философия стиля

Стиль Кузнецова — это профессиональный подход к созданию корпоративных документов, основанный на принципах:

- **Ясность** — каждое предложение несёт смысловую нагрузку
- **Структура** — логическая иерархия от общего к частному
- **Точность** — конкретные даты, суммы, ссылки на нормативные акты
- **Элегантность** — минималистичный дизайн без визуального шума

## Стандарты форматирования

Данный скилл использует единые стандарты. См. файл `formatting_standards.md`.

### Цветовая схема

| Элемент | Цвет | HEX |
|---------|------|-----|
| Заголовки всех уровней | Тёмно-синий | `#233264` |
| Основной текст | Чёрный | `#000000` |
| Шапка таблицы (фон) | Тёмно-синий | `#233264` |
| Шапка таблицы (текст) | Белый | `#FFFFFF` |

### Шрифты и размеры

| Элемент | Шрифт | Размер | Начертание |
|---------|-------|--------|------------|
| Заголовок документа | Times New Roman | 16pt | Bold, цвет #233264 |
| Заголовок раздела (H1) | Times New Roman | 14pt | Bold, цвет #233264 |
| Заголовок подраздела (H2) | Times New Roman | 12pt | Bold, цвет #233264 |
| Заголовок блока (H3) | Times New Roman | 13pt | Bold, цвет #233264 |
| Основной текст | Times New Roman | 12pt | Regular |
| Текст в таблицах | Times New Roman | 11pt | Regular |

### Параметры страницы

- **Формат**: A4 (21.0 × 29.7 см)
- **Поля**: левое 3 см, остальные 2 см
- **Межстрочный интервал**: 1.15
- **Отступ после абзаца**: 8pt
- **Выравнивание текста**: по ширине (justify)

## Структура документа

### Обязательные элементы

1. **Заголовок документа** — краткое название, отражающее суть
2. **Нумерованные разделы** — иерархическая нумерация (1., 2., 2.1., 2.2.)
3. **Таблицы с данными** — структурированная подача цифр и фактов
4. **Заключение** — краткие выводы и рекомендации
5. **Дата** — в конце документа

### Рекомендуемый порядок разделов

```
1. [Цели/Контекст] — постановка задачи
2. [План/Механизм] — способы достижения цели
3. [Анализ/Последствия] — оценка результатов
4. [Выводы и рекомендации] — итоги
```

## Таблицы

### Оформление
- Шапка: фон `#233264`, текст белый, полужирный
- Строки данных: белый фон
- Границы: Table Grid
- Числа: допускается выравнивание вправо

### Специальные строки
- Итоговые строки: полужирный текст
- Вложенные элементы: отступ 3 пробела в начале
- Отрицательные числа: в скобках, например (100)

## Списки

### Маркированные списки
- Стиль List Bullet
- Чёрные точки
- Использовать для перечисления пунктов внутри раздела

### Нумерованные подпункты
- Формат: 1) 2) 3)
- Отступ слева 1.5 см
- Использовать для пошаговых инструкций и алгоритмов

## Стиль написания

### Принципы

1. **Формальный деловой язык** — без разговорных оборотов
2. **Безличные конструкции** — «предлагается», «необходимо», «целесообразно»
3. **Точные формулировки** — избегать двусмысленности
4. **Минимум тире** — заменять на двоеточия, точки с запятой, перестройку фраз
5. **Ссылки на источники** — законы, стандарты, отчётность

### Форматирование терминов

- Сокращения расшифровываются при первом упоминании: `EuroChem Group AG (далее — ЕАГ)`
- Нормативные акты: `ПБУ 8/2010`, `МСФО (IAS) 37`, `ст. 67 ГК РФ`
- Даты: `30.11.2025`, `до 31.03.2027`
- Суммы: `405,1 млрд рублей`, `около 2,5 млрд евро`

### Акценты в тексте

- **Полужирный** — ключевые цифры, важные выводы, критические факты
- Курсив — только для дат в подписи
- UPPERCASE — только для заголовка документа (опционально)

## Создание документа

Использовать шаблон `template.py` из данного скилла.

```python
exec(open('/mnt/skills/user/kuznetsov/template.py').read())

add_title('ЗАКЛЮЧЕНИЕ')
add_subtitle('О стратегии достижения целей')
add_date('02 февраля 2026 г.')

add_section_heading('1. Стратегические цели')
add_para('Перед компанией стоят три взаимосвязанные цели:')
add_para('1.1. Первая цель.')
add_para('1.2. Вторая цель.')

add_section_heading('2. План достижения целей', size=14)
add_section_heading('Иск № 1. Название', size=12)
add_para('Описание иска.')

add_section_heading('3. Выводы и рекомендации')
add_section_heading('3.1. Первая рекомендация', size=12)
add_para('Обоснование рекомендации:')
add_bullet('первый пункт;')
add_bullet('второй пункт;')
add_bullet('третий пункт.')

add_numbered_item('1) Первый шаг алгоритма.')
add_numbered_item('2) Второй шаг алгоритма.')

data = [
    ['Показатель', 'Сумма (млрд руб.)'],
    ['Доход', '163,0'],
    ['Расход', '(20,0)'],
    ['Итого', '143,0'],
]
add_table(data)

add_signature_date('02 февраля 2026 г.')

doc.save('/mnt/user-data/outputs/document.docx')
```
